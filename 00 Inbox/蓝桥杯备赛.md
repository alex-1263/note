# 蓝桥杯编程环境
* C/C++
	* `DevC++ 5.4.0`
	* `C/C++ API`
* Python
	* `编译器:Python 3.8.6`
	* `编辑器:IDLE`(Python自带的编辑器)
* Java
	* `JDK 1.8`
	* `Eclipse-java-2020-06`
	* `Java API`
Python只能使用自带库
C/C++一定要有return 0;
<font color=#ff0f0f><b>Java不能用package!!!必须保证类名为Main!!!</b></font>
